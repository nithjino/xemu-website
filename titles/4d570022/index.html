Intro's / Cutscenes:
-----------------------
The FMV Intro and cutscenes don't render on intel integrated graphics. In-game cutscenes render correctly.

Sound:
-----------------------
In the FMV's the sound loops, I'm not sure if this only affects intel GPUs or is due to the incomplete audio pipeline.

Performance / Gameplay:
-----------------------
The game on my system has massive slowdowns throughout running mid 10's when fighting enemies with no enemies on screen, it can reach 25 fps(30 fps cap). With a better CPU, it is very playable.


Graphics:
-----------------------
The only graphical glitches I could find are Visual artefacts along screen margins, and characters can have their weapons become invisible at the start of the mission(restarting the game level fixes the issue).
